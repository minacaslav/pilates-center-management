# Pilates Center Management System

Software system for membership tracking and management in a pilates center, developed in C# using .NET and MySQL.

## Overview

This project was developed as part of the Software Design course at the Department of Software Engineering, Faculty of Organizational Sciences, University of Belgrade.
The application is based on a client-server architecture and supports management of:

* Members
* Trainers
* Membership records
* Membership types
* Training sessions
* Payments and records

## Features

* User authentication
* CRUD operations for all major entities
* Membership tracking
* Search and filtering
* Database integration
* Multi-client support using threads
* GUI integration with system operations

## Technologies

* C#
* .NET
* MySQL
* WinForms
* Object-Oriented Programming

## Software Architecture

The system uses:

* Client-server architecture
* Template Method design pattern
* Singleton design pattern
* Relational database model
